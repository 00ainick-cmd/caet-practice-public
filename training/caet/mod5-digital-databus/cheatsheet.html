The Big Three at a Glance

Attribute ARINC 429 MIL-STD-1553 RS-232
Topology 1 transmitter → up to 20 receivers, one-way only 1 Bus Controller + up to 31 Remote Terminals, two-way Point-to-point — exactly 2 devices
Direction Unidirectional Bidirectional (command / response) Bidirectional (full duplex)
Speed 12.5 kbps (low) · 100 kbps (high) 1 Mbps ~9.6 kbps (GPS NMEA) to 115 kbps
Encoding Bipolar RZ (+10V / 0V / −10V) Manchester II (self-clocking) Asymmetric ±5V to ±15V, no clock
Word length 32 bits (label 8 + SDI 2 + data 19 + SSM 2 + parity 1) 20 bits (sync 3 + data 16 + parity 1) Variable byte-oriented (start + 8 data + stop)
Cabling Twisted shielded pair (T/SP) Twisted shielded pair, 78Ω terminated each end Multi-conductor cable, DB-9 or DB-25
Redundancy None — per bus Dual-redundant (Bus A & Bus B) None
Parity Odd parity Odd parity Configurable (even / odd / none)
Typical use Air data, FMS, EFIS, autopilot — commercial/GA Flight-critical military, high-end civil (777, helos) GPS NMEA output, config / maintenance ports

Mental Models (for teaching)

ARINC 429 — Pirate Radio

One station broadcasts; many listeners tune in. Nobody asks permission; everyone just tunes to the frequency they want.

[TX]
 |
 +-→ RX1   RX2   RX3...

MIL-STD-1553 — Military Net

A Bus Controller (commander) gives orders. Remote Terminals (soldiers) only speak when called. Redundant channel if the first is cut.

[BC]
 |
BUS-A   BUS-B (dual)
 |        |
RT1 RT2 RT3...

RS-232 — Tin Cans & String

Private line between exactly two people. Want a third? Run a second string. Cannot be daisy-chained or multi-dropped.

[A] ↔ [B]
 
That's it.

ARINC 429 Word Format (32 bits, little-endian) — memorize this

| PARITY 1 | SSM 2 | DATA 19 | SDI 2 | LABEL 8 |
  bit 32   31-30   29-11   10-9    8-1

LABEL (8 bits, octal-coded) — WHAT this word is. E.g., 203 = barometric altitude, 206 = computed airspeed. Receivers filter by label.

SDI (2 bits) — Source/Destination Identifier. Lets one transmitter address specific receivers when needed.

DATA (19 bits) — the actual value (BCD or binary).

SSM (2 bits) — Sign/Status Matrix. NORMAL / NO COMPUTED DATA / FAILURE WARNING / FUNCTIONAL TEST.

PARITY (1 bit) — odd parity. Detects single-bit errors.

Troubleshooting Gold

Termination & Reflections

Signals bounce off unterminated cable ends like waves off a wall. The reflection collides with the next transmission → bit errors + "ringing" on scope.

1553:78Ω at EACH END of main cable ARINC 429:No terminator needed (RZ encoding) CAN bus:120Ω at each end

If intermittent errors appear right after avionics-bay maintenance → suspect a disturbed terminator first.

Parity & Error Detection

Parity bit is set so total "1" bits in the word match the rule (odd for 429/1553). Receiver recounts; mismatch = bit flipped in transit = word dropped.

Parity detects — does NOT correct.

A small percentage (~1-5%) of words failing parity = noise/EMI/bad shield. ALL words failing = wrong data rate or wiring fault.

Common Faults by Bus

ARINC 429:Open/shorted pair, swapped polarity, missing/floating shield ground 1553:Missing terminator, coupling transformer fault, bad BC, shorted stub RS-232:Wrong pinout (TX/RX swap), ground loop, cable too long (>50 ft)

Test Equipment Ladder

Step 1:DMM — continuity, shield ground, basic voltages Step 2:Scope — waveform shape, ringing, levels Step 3:Bus analyzer — protocol decode, parity, label traffic

Escalate from simplest to most complex. Don't skip to the analyzer when a DMM would find the fault.

What the CAET Loves to Test

  • Which bus is unidirectional? → ARINC 429
  • Which bus uses command/response? → 1553
  • Which bus is point-to-point only? → RS-232
  • What is the data rate of 1553? → 1 Mbps
  • What terminates a 1553 bus? → 78Ω each end
  • How many receivers on one ARINC 429 transmitter? → 20 max
  • First 8 bits of an ARINC 429 word? → Label (octal)
  • Which bus uses Manchester II encoding? → 1553
  • What does the parity bit do? → Detect (not correct) single-bit errors
  • Ringing on 1553 scope + recent maintenance = terminator fault
Teaching shortcut: If a student can recite the three mental models (pirate radio / military net / tin cans & string) AND the termination rule (78Ω / each end / 1553 only), they will get 70% of CAET Module 5 questions right.
